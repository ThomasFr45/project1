Langages travaillés :
Pendant votre formation, vous approfondirez vos connaissances en HTML et CSS, mais vous apprendrez aussi les bases de JavaScript ainsi que PHP !
Après vos deux premières semaines, on vous demandera de choisir le langage avec lequel vous souhaitez continuer la formation, qu'allez vous choisir ?
Workshops :
Au cours de la formation, différents ateliers seront mis en place afin de mettre en pratique ce qui à été vu lors des cours.
Ces derniers peuvent se dérouler seul ou en groupe. Notez qu'il est possible de les refaire à n'importe quel moment !
Quêtes :
Les quêtes représentent une grosse partie du cursus, vous en aurez des nouvelles toutes les semaines.
Ces dernières, vous permettront de voir certaines notions, et de les mettre en pratique (du moins essayer) avant le cours les concernants. Il est donc TRES important de les faire sérieusement !
Projets :
Entre vos cours, workshops et vos quêtes, on vous demandera de réaliser des projets.
Ces derniers se réaliseront en groupes, afin de vous permettre d'apprendre les un des autres.
Il sera donc très important de bien s'organiser et discuter avec ses collègues.
Attentes :
Et oui, la Wild Code School, ce n'est pas une promenade, on attend beaucoup de vous !
Voici une petite idée de ce qui vous attends si vous choisissez de continuer le JavaScript :

Semaine 4 : Introduction POO (Programmation Orientée Objets)
Semaine 5 Checkpoint 1 HTML / CSS / JS
Semaine 6 : Start React + Projet 2 sur React + call API + Introduction Backend Node.JS Express
Semaine 10 : Rendu Projet 2
Semaine 11 : Checkpoint 2 React + API // Hackathon 1
Semaine 12 : Introducion Base de données + Node.Js Express MVC
Semaine 13 : Start Projet Client Fullstack
Semaine 16 : Checkpoint 3 (Node.JS)
Semaine 17 : Hackathon 2
Semaine 20 : Final Checkpoint (React + Node) + Livraison Projet Client
